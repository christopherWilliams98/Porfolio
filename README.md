# Project Portfolio

This repository contains a selection of academic, research, and software development projects I have worked on, covering data science, simulation, mobile development, computer graphics, and systems integration.

## Projects

### [CelluloEthicalGame](./CelluloEthicalGame)
An interactive game project exploring ethical decision-making through gameplay and user interaction.

### [Radiated_Power_Bolometric_estimation](./Radiated_Power_Bolometric_estimation)
A data analysis and machine learning project focused on estimating radiated power from bolometer measurements in experimental physics datasets.

### [Rayleigh_Scattering_CGI](./Rayleigh_Scattering_CGI)
A computer graphics project demonstrating Rayleigh scattering effects, used to simulate atmospheric light scattering and sky coloration.

### [SBB_Large_Scale_Data](./SBB_Large_Scale_Data)
A large-scale data processing project using Swiss railway data to explore route planning with risk aware optimizations, and transport data analysis.

### [ScribeWiz_app](./ScribeWiz_app)
An Android music transcription application that records audio from an instrument and converts it into musical notation and XML files.

### [SitaWare_C2SIM_Gateway](./SitaWare_C2SIM_Gateway)
A systems integration project involving a gateway between SitaWare and C2SIM-style command-and-control simulation interfaces.
