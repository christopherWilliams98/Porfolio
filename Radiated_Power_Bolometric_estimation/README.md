# Project Readme

## Overview

This project involves analyzing and processing data related to radiated power, as captured by bolometers in various experimental setups. The repository is structured to facilitate easy setup and execution of the analysis code.

## Getting Started

### Prerequisites

Before running the project, ensure that you have the following prerequisites installed:

- Python 3.10.11
- Necessary Python modules as listed in the first cell of the notebook.
    You can run pip install -r requirements.txt to easily install these.

### Installation

1. **Clone the Repository**

### Dataset Setup


- **Download the Dataset:**
You need to download either `dataset_bolo_radiated_power.pickle` or `dataset_bolo_radiated_power.h5`. These files contain the necessary data for the project.


- **Configure the Dataset:**
In your script or Jupyter notebook, set the `use_pickle` flag:
- Set `use_pickle = True` if you are using the `.pickle` file.
- Keep `use_pickle = False` if you are using the `.h5` file.

## Usage

After installation, you can run the scripts or Jupyter notebooks present in the repository. Make sure to follow the comments and instructions within the code for optimal results.
